웹퍼블리싱 공모전 포스터 2D Grapic Design

Overview

웹퍼블리싱(모바일 웹 사이트 제작) 공모전을 소재로 안내 포스터를 제작하였습니다. 뮤지컬/연극/공연/전시회/박람회 등을 소개하는 웹 제작으로 포스터에서도 관련 오브젝트들을 통해 직관적으로 주제를 표현하였고 타이포에 접목한 일러스트 작업을 하였습니다.
    • 작업 기여도
    • 개인작업 100%
    • 작업 프로그램

Analysis

공모전 TEXT
  • 모바일 웹 사이트 제작공모전
  • 21.11.01 ~ 21.11.10 오후5시 까지
  • 웹퍼블리싱
  • 뮤지컬, 연극, 공연, 전시회, 박람회 등을 소개하는 모바일 전용 웹페이지

    인트로, 공연소개 및 안내, 등장인물소개, 공연일정안내 및 요금, 예약하기, 오시는 길 등 주제에 맞는 모바일용 웹페이지로 10page이내 제작

  • 21.11.10 오후 6시 ~ 21.11.11 오후6시
  • 온라인투표(30점) + 담당강사 외 심사위원(20점) + 담당강사(50점)
  • 2021.11.12

    • 우수상(2명) : 5만원 상당의 상품권
    • 장려상(5명) : 2만원 상당의 상품권
공모전의 결과물인 웹퍼블리싱을 '문자 코드를 이용한 새로운 공간 창조'로 해석하여, 입체효과를 준 일러스트를 작업하였습니다.
입체효과가 적용된 글자는 큰 전시공간을 상징하며 세부주제에 해당하는 오브젝트들을 조화롭게 조합하였습니다.
(티켓박스,입퇴장로,액자,수족관,관람좌석)
마지막으로 선적인 요소들을 더하여 평면에서 가상현실로 변화되는 모습을 나타냈습니다.

Work Process
