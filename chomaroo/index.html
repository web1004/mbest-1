All about ChoMaroo's Fragrance

자연을 담아낸 초마루의 다양한 향을 만나보세요.

  • 플라워마켓

    Flower Market

    Story

    오직, 초마루만의 블랜딩 기법으로 만들어낸 대표 시그니처향으로 인워적이지않고, 자연스러운 발향을 선사합니다. 여러분의 공간에 '자연의 향'을 더하세요. 초마루의 플라워마켓 향은 마치 신선한 풀내음과 꽃향기로 어우러진 꽃시장에 들어선 듯한 느낌을 전해주는 초마루의 시그니처향으로 고객들께 가장 사랑받는 향입니다.

    Note
    • TOPIVY GREEN / PEEL GREEN / CARDAMOM / EUCALYPTUS / LAVANDIN / GREEN EARTHY
    • MIDDLEROSE / FREESIA / JASMINE/LILAC
    • BASESWWEET BALSAM / GREEN WOOD / CINNAMON/CLOVE LEAF
  • 루체떼

    Lucete

    Story

    루체떼는 초마루의 신규 여섯번째 향으로, 대나무의 '마디'처럼 각 '노트'별 색다른 향으로 여러분들께 다가갑니다. 이탈리안 베르가못 향을 포함하여 카사블랑카 릴리, 일랑일랑, 그레이 앰머, 샌달 우드의 조화로 대나무향이 물씬 풍겨오는 새로운 초마루의 루체떼향으로 깊은 힐링을 시작하세요.

    Note
    • TOPBOIS DE ROSE / WATERMELON
    • MIDDLEJASMINE / FREESIA / WATERY ORCHILD
    • BASELIGHT BALSAM / MUSK / SPICY MUSK
  • 시트론

    Citron

    Story

    시트론은 자연을 담은, 초마루의 다섯번째 향으로 청량한 스파클링 같은 샴페인향과 따뜻한 흙내음 우디향의 조화로 시작합니다. 가볍고 상쾌한 레몬 껍질의 상큼함과 더불어 묵직함이 더해지는 패출리의 흙내음까지 맡아볼 수 있어 마음이 편해지는 매력적인 향을 선사합니다.

    Note
    • TOPLEMON / GREEN APPLE / PINK GRAPEFRUIT / ORANGE
    • MIDDLEFRESSIA / ROSE / ORANGE FLOWER
    • BASEMUSK
  • 네추럴우드

    Natural Wood

    story

    초마루의 내추럴우드 향은 묵직한 대자연 속의 나무들을 떠올리게 하는 숲 속의 향을 담았습니다. 공간을 차분하고 안정적인 분위기로 만들어주는 향 입니다. 초마루의 블렌딩 기번으로 만들어 '자연'그대로의 향으로 우아한 나무향과 기대 밖의 신선한 향으로 자연이 주는 풍부한 향을 지금 경험하세요.

    Note
    • TOPORANGE / GINEGER / NUTMEG
    • MIDDLEROSE / OLIBANUM
    • BASECEDARWOOD / SANDALWOOD / PATCHOULI / VETIVER
  • 화이트사본

    White Savon

    Story

    초마루의 화이트사본 향은 높고 푸른 하늘에 은은하게 퍼지는 맑고 투명한 물방울과 같이 상쾌하면서 투명한 향이 그윽하게 자리잡습니다. 자스민 민트, 머스크의 조화가 이루어져 깨끗한 비누를 사용한 것과 같은 달달함과 섬유유연제 같은 부드러운 향은 물론, 포근함이 느껴지는 대표 디퓨져입니다.

    Note
    • TOPAPPLE / LEMON / PLUM
    • MIDDLEFREESIA / JASMINE / MUGUET / ROSE
    • BASEMUSK / AMBER / WATERY
  • 더 리프

    The Leaf

    Story

    상쾌한 나뭇잎을 연상시키는 아로마향으로 촉촉한 이슬과 만난 풀잎들 사이, 자연 본연의 향을 담아냈습니다. 플랜테리어와 잘 어울리는 초마루의 대표 디퓨저입니다. 디퓨저 하나로 시작되는 기분전환, 건강한 나뭇잎 속에 있는 듯한 '더 리프' 향으로 전해지는 힐링의 세계. 나뭇잎 향기로 지친 몸과 마음의 힐링을 시작하세요.

    Note
    • TOPORANGE / MANDARIN / GRAPEFRUIT / LYCHEE / GUAVA
    • MIDDLEMUGUET / JASMINE / OSMANTHUS
    • BASEMUSK / VETIVER / BALSAM

How to Use The Diffuser

디퓨저, 이렇게 사용하세요.

  • 디퓨저 액을 디퓨져 병에 1/3정도 채워줍니다.
  • 디퓨저 병에 섬유스틱을 꽂아줍니다.
  • 원하는 발향 정도에 따라 개수를 조절해주세요.
  • 주기적으로 섬유스틱을 뒤집거나 교체해 주면 발향을 오래 즐길 수 있습니다.
  • 공간의 온도, 습도, 공기 순환 등 환경에 따른 발향력의 차이가 생길 수 있습니다.
Q. 디퓨저 향이 잘 나지 않아요. expand_circle_down
디퓨저 스틱은 온도, 바람, 습도 등 환경적인 요인에 의해서 마르는 현상이 발생됩니다. 특히 디퓨저를 창문 주변에 두고 사용하시면 자외선과 바람의 영향으로 스틱이 금방 마르고 먼지가 쉽게 쌓여 발향력이 약해질 수 있습니다. 또한 화장실에 두고 사용하실 경우 습도가 높아져 수분 접촉으로 인해 발향력이 약해집니다.
Q. 발향이 더 잘되게 하고 싶어요. expand_circle_down
디퓨저 스틱은 발향 강도를 조절해주는 소모품으로 처음 사용하실 때 1~2개 정도의 스틱을 꽂고 향을 맡어보신 후 취향에 따라 추가해 주시면 됩니다.섬유스틱은 향을 끌어올리는데 1~2일 정도의 시간이 소요되며, 우드스틱이나 플라워스틱은 더 오랜 시간이 걸릴 수 있습니다. 저희 초마루에서 추천드리는 스틱 사용 기간은 약 1개월이며 오일의 양이 줄어들지 않을 때 스틱을 교체해주시면 됩니다.
Q. 가죽, 목재 가구에 디퓨저 액이 묻었어요. expand_circle_down
모든 오일 액은 가죽이나 목재, 플라스틱, 의류 등에 묻었을 시 변색 및 탈색, 훼손의 우려가 있습니다. 디퓨저 액이 묻은 것을 발견하는 즉시 천이나 티슈 등으로 닦아내주시기 바라며 디퓨저 사용 시 받침 사용을 권장합니다.
